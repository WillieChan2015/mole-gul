# Docs 目录索引

本目录包含 Mole GUI 项目的所有文档。

## 目录结构

```
docs/
├── README.md                          # 本文件：目录索引
├── CHANGELOG.md                       # 版本历史和重要变更
├── DOCS_INDEX.md                      # 详细文档索引和说明
└── superpowers/                       # 开发过程文档
    ├── architecture/                  # 架构设计
    │   └── implementation_plan.md     # 详细实施方案和架构设计
    ├── roadmap/                       # 路线图和规划
    │   └── next_phase_plan.md         # 下一阶段规划和任务清单
    ├── compliance/                    # 合规和标准
    │   └── open_source_checklist.md   # 开源项目标准检查清单
    ├── testing/                       # 测试和验证
    │   └── phase0_results.md          # Phase 0 验证结果
    ├── plans/                         # 功能规划
    │   ├── 2026-06-11-data-visualization.md
    │   ├── 2026-06-11-i18n.md
    │   └── 2026-06-11-tailwindcss-integration.md
    └── specs/                         # 设计规格
        ├── 2026-06-11-data-visualization-design.md
        ├── 2026-06-11-i18n-design.md
        ├── 2026-06-11-i18n-requirements.md
        └── 2026-06-11-tailwindcss-integration-design.md
```

## 文件说明
